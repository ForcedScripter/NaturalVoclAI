# MINISTROS — Next.js app

This folder is the **main application**. All install and run commands should be executed here.

## Run locally

```bash
# From repository root, enter this folder first:
cd NaturalVoclAI

npm install
# Create .env.local (see parent README for variables)
npm run dev
```

Open **http://localhost:3000**

Full documentation (repo layout, env vars, backend, deployment, troubleshooting) is in the **[root README](../README.md)**.
